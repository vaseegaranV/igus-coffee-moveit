# COAR-ROS2-Workspace
This is the ROS2 Jazzy workspace for Cobot-On-A-Rail project to assess the security of Digital Twins

Launching Gazebo: ros2 launch igus_rebel_moveit_config igus_rebel_simulated.launch.py 
Launching Rviz: ros2 launch igus_rebel_moveit_config moveit_rviz.launch.py
Launching Pick and Place: ros2 launch igus_rebel pick_and_place.launch.py
